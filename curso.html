Materiales del Curso

Temas Aprendidos

  • HTML5: Uso de etiquetas para crear la estructura de una página web.
  • CSS3: Aplicación de colores, estilos, diseño y animaciones.
  • Enlaces Relativos: Navegación entre páginas internas del sitio web.
  • Enlaces Absolutos: Enlaces hacia páginas externas usando URL completa.
  • Formularios: Uso de inputs, botones, textos y envío de datos.
  • Tablas: Organización de datos en filas y columnas.
  • GitHub: Publicación y almacenamiento de proyectos web online.
  • Responsive Design: Adaptación de la página para celulares y tablets.

Lo aprendido en el curso

En este curso aprendí a crear páginas web modernas utilizando HTML y CSS, además de conectar varias páginas entre sí mediante enlaces.

También aprendí a subir proyectos a GitHub y mejorar el diseño visual de una página web profesional.
